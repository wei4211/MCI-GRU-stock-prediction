# MCI-GRU：基於多頭交叉注意力與改良門控循環單元的股票預測模型

> 東吳大學資料科學系｜金融科技與智能交易｜期末報告  
> 指導老師：鄭宏文 副教授

---

## 專案簡介

本專案實作並改良 **MCI-GRU** 模型，以台灣股市（鴻海、台積電、華邦電、國巨）為實證標的，透過每日滾動式（Rolling-Expanding）重訓與回測，評估模型在真實台股環境中的預測能力與投資績效。

相較於原論文的「多日排名預測」設定，本研究改為**預測下一交易日報酬率**，並搭配多項金融績效指標進行全面評估。

---

## 模型架構

MCI-GRU 由四個核心模組組成：

| 模組 | 功能 |
|------|------|
| **Module A**：改良型 GRU | 以注意力機制取代傳統重置門，強化長期時序依賴的捕捉能力 |
| **Module B**：圖注意力網路（GAT） | 根據股票歷史報酬的皮爾森相關係數建立關聯圖，捕捉橫截面共動關係 |
| **Module C**：多頭交叉注意力 | 引入可學習的潛在市場向量，建模不可直接觀測的市場情緒與資金輪動 |
| **Module D**：預測與損失計算 | 融合四路特徵後接 GAT 進行最終報酬預測，以 MSE 為損失函數 |

## 預測流程（Rolling-Expanding）

針對測試區間內的每一個交易日 D：

1. 以 D 回推決定驗證集範圍（固定兩年長度）
2. 訓練集終點為驗證集起點前一日，並向後 Expanding 擴增
3. 取前 60 個交易日的特徵序列與股票關聯矩陣作為模型輸入
4. 每日重新初始化並訓練 MCI-GRU（3–5 epochs，附 early stopping）
5. 輸出該日各股票的預測報酬率，作為交易訊號（正→買進、負→賣出）
6. 時間窗向前推進，重複上述步驟

---

## 評估指標

**金融績效指標（越高越好）**：ARR（年化收益率）、ASR（年化夏普比率）、CR（卡爾瑪比率）、IR（資訊比率）、勝率、累積報酬

**金融風險指標（越低越好）**：AVoL（年化波動率）、MDD（最大回撤）

**預測準確性指標**：MSE（均方誤差）、MAE（平均絕對誤差）

## 實驗結論

1. **策略累積報酬多數落後買進持有與大盤**：模型在多頭順風市況下較能跟上，但在高噪音或趨勢轉折情境下容易失真。
2. **拉長訓練期主要改善穩定性，不保證提升報酬**：長訓練後曲線更平滑、極端崩跌較少，但超額報酬的提升效果有限。
3. **勝率約介於 43%–51%**：樣本報酬在長訓練下略有改善；對數報酬的勝率無一致性提升。
4. **樣本報酬 + 長訓練整體一致性最佳**；對數報酬結果較不穩定，呈現少數標的好但不可複製的情況。
5. **華邦電表現相對最弱**：該股 regime shift 明顯，日頻模型難以應對急漲急跌段，MDD 仍偏高。
